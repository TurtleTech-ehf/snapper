#+title: Word-Level Diff (latexdiff-style)
#+options: num:nil toc:nil

* Overview

=snapper wdiff= compares two files at the word level within sentences, highlighting exactly which words changed.
This gives latexdiff-style output but works for any format, not just LaTeX.

* Basic usage

#+begin_src bash
snapper wdiff paper_v1.tex paper_v2.tex
#+end_src

Terminal output (default) uses ANSI colors: red strikethrough for deleted words, green bold for added words.

* Output formats

Control the markup style with =--output-format=:

#+begin_src bash
# Plaintext (default for piping)
snapper wdiff --output-format plaintext old.org new.org
# Output: This approach [-dramatically-] {+significantly+} reduces noise.

# LaTeX (latexdiff-compatible)
snapper wdiff --output-format latex old.tex new.tex
# Output: This approach \DIFdel{dramatically} \DIFadd{significantly} reduces noise.

# Markdown
snapper wdiff --output-format markdown old.md new.md
# Output: This approach ~~dramatically~~ **significantly** reduces noise.

# Org-mode
snapper wdiff --output-format org old.org new.org
# Output: This approach +dramatically+ *significantly* reduces noise.

# Typst
snapper wdiff --output-format typst old.typ new.typ
# Output: This approach #strike[dramatically] #highlight[significantly] reduces noise.
#+end_src

* LaTeX PDF output

Generate a compilable LaTeX document with change markup:

#+begin_src bash
snapper wdiff --output-format latex paper_v1.tex paper_v2.tex > diff.tex
#+end_src

The =\DIFadd{}= and =\DIFdel{}= commands are compatible with latexdiff's DIF style.
Add this preamble to compile:

#+begin_src latex
\usepackage{color}
\providecommand{\DIFadd}[1]{{\protect\color{blue}\textbf{#1}}}
\providecommand{\DIFdel}[1]{{\protect\color{red}\sout{#1}}}
\usepackage[normalem]{ulem}
#+end_src

* Comparison with latexdiff

| Feature | latexdiff | snapper wdiff |
|---------+-----------+---------------|
| Formats | LaTeX only | Org, LaTeX, Markdown, RST, typst, plaintext |
| Installation | Perl + TeX Live | Single binary |
| Speed | Slow (Perl) | Fast (Rust) |
| Output | LaTeX only | 6 formats (terminal, plaintext, LaTeX, Markdown, Org, typst) |
| Sentence awareness | No | Yes (sentence-level + word-level) |
| Git integration | Manual | =snapper git-diff --word-level= |

* Integration with sdiff and git-diff

The =--word-level= flag on =snapper sdiff= and =snapper git-diff= enables word-level highlighting within changed sentences (planned, currently wired but rendering through the dedicated =wdiff= subcommand).
