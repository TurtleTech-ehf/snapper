#+title: CI Enforcement
#+options: num:nil toc:nil

* GitHub Action (recommended)
:PROPERTIES:
:CUSTOM_ID: github-action
:END:

The easiest way to enforce semantic line breaks in CI.
Add to your workflow:

#+begin_src yaml
- uses: TurtleTech-ehf/snapper@v0.6.0
  with:
    files: '**/*.org **/*.tex **/*.md'
#+end_src

This installs snapper and runs =--check= on the specified files.

For GitHub Code Scanning integration (SARIF annotations on PRs):

#+begin_src yaml
- uses: TurtleTech-ehf/snapper@v0.6.0
  with:
    files: '**/*.org **/*.tex **/*.md'
    sarif: 'true'
#+end_src

* Pre-commit hook
:PROPERTIES:
:CUSTOM_ID: pre-commit
:END:

Add to =.pre-commit-config.yaml=:

#+begin_src yaml
- repo: https://github.com/TurtleTech-ehf/snapper
  rev: v0.6.0
  hooks:
    - id: snapper
#+end_src

This runs =snapper --in-place= on changed files matching =.org=, =.tex=, =.md=, and =.txt=.

* Manual GitHub Actions
:PROPERTIES:
:CUSTOM_ID: manual-github-actions
:END:

If you prefer manual setup over the composite action:

#+begin_src yaml
- name: Install snapper
  run: cargo binstall -y snapper-fmt

- name: Check formatting
  run: snapper --check paper.org sections/*.org
#+end_src

For structured output:

#+begin_src yaml
- name: Check with SARIF
  run: snapper --check --output-format sarif **/*.org > snapper.sarif || true

- uses: github/codeql-action/upload-sarif@v3
  with:
    sarif_file: snapper.sarif
#+end_src

* Preview changes in CI
:PROPERTIES:
:CUSTOM_ID: preview-changes
:END:

Use =--diff= to show what would change:

#+begin_src yaml
- name: Show formatting diff
  run: snapper --diff paper.org sections/*.org
#+end_src

* GitLab CI
:PROPERTIES:
:CUSTOM_ID: gitlab-ci
:END:

#+begin_src yaml
format-check:
  image: rust:latest
  script:
    - cargo binstall -y snapper-fmt
    - snapper --check **/*.org **/*.tex **/*.md
  allow_failure: false
#+end_src

* Makefile integration
:PROPERTIES:
:CUSTOM_ID: makefile
:END:

#+begin_src makefile
.PHONY: fmt fmt-check

fmt:
	snapper --in-place paper.org sections/*.org

fmt-check:
	snapper --check paper.org sections/*.org
#+end_src
